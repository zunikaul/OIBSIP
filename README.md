# Completed Tasks

## Task 1: Network Scanning Using Nmap

### Description

In this task, I performed network scanning using Nmap to understand how systems are discovered during the reconnaissance phase of cybersecurity testing.

The scan was used to identify open ports, running services, and basic information about the target system.

### Tool Used

- Nmap

### Concepts Learned

- Network reconnaissance
- Port scanning
- Service detection
- Basic vulnerability identification

### Tool Used

# Task 3: Network Traffic Analysis Using Wireshark

### Description

In this task, I captured and analyzed network packets using Wireshark.

The captured traffic was further analyzed using Python and Scapy to identify protocols, source IP addresses, destination IP addresses, and generate a basic network analysis report.

### Tools Used

- Wireshark
- Python
- Scapy

### Work Performed

- Captured network traffic
- Studied packet information
- Identified protocols in communication
- Analyzed source and destination addresses
- Generated an automated report using Python

### Concepts Learned

- Packet analysis
- Network protocols
- PCAP file analysis
- Network monitoring

# Task 4: Phishing URL Detection System

### Description

In this task, I developed a Python-based tool to analyze URLs and identify suspicious characteristics commonly found in phishing links.

The program checks different URL parameters and generates a risk score based on detected indicators.

### Tools Used

- Python
- Regular Expressions
- URL Parsing

### Features Implemented

- HTTPS checking
- URL length analysis
- Suspicious keyword detection
- Special character checking
- IP address URL detection
- Risk score calculation

### Concepts Learned

- Phishing attack analysis
- URL-based threat detection
- Security automation
- Basic risk assessment
